# PhantomChat.UI

## FE for the https://github.com/h128/PhantomChat

Vite + React + TypeScript starter configured with:

- React Compiler
- Tailwind CSS v4
- Redux Toolkit
- React Router

## Scripts

- `npm run dev`
- `npm run build`
- `npm run lint`
- `npm run preview`
